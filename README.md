\# Court Booking System



A web-based court booking system for institute sports (Badminton, Squash, Tennis).



\## Tech Stack

\- React.js (Vite)

\- Tailwind CSS

\- shadcn/ui

\- React Router



\## Features

\- View available courts and time slots

\- Book a court via a booking form

\- View booking status (Pending / Approved / Rejected)

\- Admin dashboard to approve or reject bookings



\## Assumptions

\- No real backend is used; all data is stored in React state (mock data)

\- Any student can access the admin dashboard via the /admin route

\- Bookings reset on page refresh as there is no database



\## Setup Instructions

1\. Make sure Node.js is installed (https://nodejs.org)

2\. Clone or download this repository

3\. Open a terminal and run: npm install

4\. Then run: npm run dev

5\. Open your browser and go to: http://localhost:5173

6\. For the Admin Dashboard, go to: http://localhost:5173/admin
